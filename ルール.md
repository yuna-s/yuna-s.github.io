# 毎日ブログを書くルール

Claudeと一緒に記事を書くときのルール。迷ったらここを見る。

---

## 記事を書くときの流れ

1. `posts/_template.html` をコピーして新しいファイルを作る
2. ファイル名は `YYYY-MM-DD-キーワード.html`（例: `2026-04-01-morning-routine.html`）
3. 記事を書く（EN・JAどちらでもOK、テンプレートに沿って）
4. `index.html` に記事カードを追加する（後述）
5. プレビューで確認してからpush

---

## ファイル命名規則

```
posts/YYYY-MM-DD-英語キーワード.html
```

- 日付は投稿日
- キーワードはハイフン区切りの英語（スペースなし）
- 例: `2026-04-05-first-cup-of-coffee.html`

---

## タグ（4種類のみ）

| タグ | 内容 |
|------|------|
| `daily life` | 日常・暮らし・感じたこと |
| `tech` | 技術・開発・ツール |
| `books` | 本・読書 |
| `writing` | 書くこと・ブログ運営 |

新しいタグを増やすときは index.html のフィルターボタン・タグクラウド・i18n翻訳も更新する。

---

## index.html に記事カードを追加する手順

### 1. ブロググリッドにカードを追加（新しい記事を一番上に）

```html
<a class="post-card" href="posts/YYYY-MM-DD-title.html" data-tags="タグ名"
   data-content="検索用キーワード（本文の重要な単語をスペース区切りで）">
  <div class="post-meta">
    <span class="post-date" data-i18n="dateN">Month DD, YYYY</span>
    <span class="post-tag" data-tag="タグ名" data-i18n="tag-XXX">🌿 daily life</span>
  </div>
  <div class="post-title" data-i18n="postN-title">英語タイトル</div>
  <div class="post-excerpt" data-i18n="postN-excerpt">英語抜粋文</div>
  <div class="post-read" data-i18n="read-more">Read more →</div>
</a>
```

### 2. i18n翻訳オブジェクトに追加（en・ja 両方）

```js
// en セクション
'dateN':       'Month DD, YYYY',
'postN-title': '英語タイトル',
'postN-excerpt': '英語抜粋文',

// ja セクション
'dateN':       'YYYY年M月D日',
'postN-title': '日本語タイトル',
'postN-excerpt': '日本語抜粋文',
```

### 3. サイドバーの Recent Posts を更新

最新4件だけ表示。古いものを一番下に追加して、5件目以上は削除。

### 4. Stats の記事数を更新

```html
<span class="stat-num">5</span>  ← 記事数
```

---

## 言語ルール

- **記事本文**：英語で書く（日本語版は任意・後から追加OK）
- **UIテキスト**：EN・JA 両方必須（index.html の i18n オブジェクトに追記）
- **ファイル名・タグ値**：常に英語
